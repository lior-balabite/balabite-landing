# BalaBite.ai — Character & Illustration Generation Guide
### Updated March 19, 2026

---

## Reference & Style

**Primary reference**: Fig.security — low-poly/papercraft 3D illustration. Warm, tactile, handmade. Pixar meets children's book, for adults. NOT robotic, NOT neon, NOT corporate.

**Palette for all prompts**:
```
warm amber (#F59E0B), terracotta (#C2410C), sage green (#65A30D),
cream white (#FAF5EE) background, warm purple (#7C3AED), soft coral (#FB923C),
muted teal (#0D9488)
```

**Style suffix for all prompts** (append to every prompt):
```
low-poly papercraft style, warm color palette, cream background, soft shadows,
Pixar-meets-papercraft aesthetic, minimal detail, no text --style raw --no text words letters
```

---

## Recommended Tool Pipeline

| Step | Tool | Why |
|---|---|---|
| 1. Explore concepts | **ChatGPT (GPT Image 1.5)** | Free, fast, best prompt adherence. Iterate in conversation. |
| 2. Consistency | **Neolemon** | Lock character, generate unlimited poses/scenes. Purpose-built for this. |
| 3. Vectors | **Recraft V4 Pro** | Recreate finals as editable SVG for the website. Only tool with native vector. |
| 4. Scenes | **Midjourney V7** with --sref | Best composition for multi-character hero scenes. |
| 5. Refinement | **Figma** | Clean vectors, adjust colors, export production assets. |
| 6. Animation | **Kling 3.0** | Animate characters for landing page video (later). |

**Quick start**: Open ChatGPT, paste a character prompt below, iterate until you love it. Then move to Neolemon/Recraft for consistency and vectors.

## GENERATION TIPS

1. **Start with The Pulse** — it's the centerpiece. Get this right first.
2. **In ChatGPT**: iterate in one conversation thread. "Make it rounder." "More warmth." "Less robotic." The conversational flow is the advantage.
3. **Once you love a character**: screenshot it and use it as a style reference for all subsequent characters (--sref in Midjourney, style lock in Recraft, reference upload in Neolemon).
4. **Test at small sizes**: zoom out to 64px, 32px. If unrecognizable, simplify.
5. **The cream background is non-negotiable** — it's what separates this from every dark-mode SaaS site.
6. **All characters should wear a subtle shared element** — same color apron, same pattern, same accessory. Like Fig's checkered outfits. This creates visual unity.
