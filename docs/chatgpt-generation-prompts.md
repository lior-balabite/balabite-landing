## After Generating: Evaluation Criteria

Rate each result on these 5 questions:

| Question | What You're Testing |
|----------|-------------------|
| 1. "Would a restaurant owner put this on their wall?" | Emotional appeal, craft quality |
| 2. "Can I tell what this is at 32px?" | Scalability, icon potential |
| 3. "Does this feel like a restaurant, not a tech company?" | Industry authenticity |
| 4. "Would I screenshot this and send it to someone?" | Shareability, memorability |
| 5. "Does this make me feel something?" | Emotional resonance — the only metric that matters |

---

## Iteration Tips for ChatGPT

- If a result is close but not right, say: "Keep the same style but make it [warmer / simpler / more textured / less cartoonish]"
- If you love one element, say: "Take the eyes from image 1 and the texture from image 2"
- To push toward more premium: "Make it feel like a Pixar concept art piece, not a children's book illustration"
- To push toward simpler: "Reduce detail by 50%. It should work as a 64px icon"
- To get consistency across characters: "Use the exact same art style, lighting, and texture as [paste previous prompt]"
