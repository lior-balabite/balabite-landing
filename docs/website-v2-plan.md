## DESIGN PRINCIPLES (from Fig analysis + design bible)

1. **One visual world** — Cream background, papercraft characters, warm amber glow. Never break the world.
2. **Lighting IS the story** — Problem sections can be darker, solution sections are light/warm.
3. **Show the story, not the speedometer** — Show moments of value, not dashboards.
4. **One CTA repeated** — "Put AI to Work" everywhere. Single action.
5. **Zero jargon in headlines** — Technical language in body copy only.
6. **Restraint = premium** — Fig has exactly 4 CSS animations. Don't over-animate.
7. **Characters > icons** — The brain characters are the brand. Use them on about page, onboarding, loading states — everywhere except the conversion funnel (per roundtable decision).
